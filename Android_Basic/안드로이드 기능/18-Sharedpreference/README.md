# SharedPreference
  
### SharedPreference
- (key-value 방식으로) 데이터 저장 가능
- 휴대폰에 데이터를 저장한다
- 데이터는 앱 밑에 있다. → 앱을 삭제하게 되면 데이터도 삭제된다.
- 사용자의 기호와 같은 간단한 데이터를 저장해야한다.
- CRUD
  - Create
  - Read
  - Update
  - Delete
