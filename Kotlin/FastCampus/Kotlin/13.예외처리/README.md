# 12.예외처리

### 예외처리
- 예외가 발생하는 부분 → try{}
- 예외를 처리하는 부분 → catch{}
```kotlin
try{
	// 코드 -> 예외발생 가능한 코드
}catch(변수명: 예외타입){
	// 예외를 처리하는 코드
}finally{
	// 이러나 저러나 무조건 실행되어야 코드
}
```
- try문에서 A라는 예외가 발생하면 catch문에서 A라는 예외를 처리하겠고 명시적으로 적어붜야 한다. 그렇지 않으면 오류가 발생한다.
- 모든 종류의 예외를 포함하는 예외타입 → Exception
- try, catch는 독립적으로 사용할 수 없다.
